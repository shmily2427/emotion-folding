# 快速开始指南

## 本地运行

### 方法一：使用 Python（推荐）

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问：`http://localhost:8000`

### 方法二：使用 Node.js

```bash
# 安装 http-server（如果未安装）
npm install -g http-server

# 运行服务器
http-server -p 8000
```

然后在浏览器中访问：`http://localhost:8000`

### 方法三：使用 VS Code Live Server

1. 安装 VS Code 的 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 操作说明

### 视角控制
- **鼠标拖动**：旋转视角
- **鼠标滚轮**：缩放场景
- **母亲视角按钮**：切换到母亲视角（略俯视，母亲在画面前侧）
- **孩子视角按钮**：切换到孩子视角（稍远，看到母亲在远处平台上）
- **情感视角对齐按钮**：切换到特殊角度，让断桥在视觉上连接

### 机关操作
- **触发机关：引路**：当视角对齐后，点击此按钮可以让桥段移动，并触发母子会合动画

### 记忆保存
- **保存此刻记忆**：将当前画面保存为 PNG 图像文件

## 核心玩法流程

1. **初始状态**：母亲和孩子分别站在两侧的平台上，中间有一座断桥
2. **切换视角**：点击"情感视角对齐"按钮，相机平滑移动到特殊角度
3. **视觉连接**：在情感视角下，断桥在屏幕上看似连接成一条路
4. **触发机关**：点击"触发机关：引路"按钮，桥段会轻微移动，强化视觉错觉
5. **母子会合**：母亲和孩子会走向桥的中心，完成会合
6. **保存记忆**：点击"保存此刻记忆"按钮，将这一时刻保存为图片

## 技术说明

- 项目使用 ES6 模块化设计
- Three.js 通过 CDN 加载（需要网络连接）
- 如果模型文件（mother.glb, child.glb）不存在，会自动使用占位几何体
- 所有功能都可以在没有模型文件的情况下正常运行

## 故障排除

### 场景不显示
- 检查浏览器控制台是否有错误
- 确保网络连接正常（Three.js 从 CDN 加载）
- 尝试刷新页面

### 模型不显示
- 检查 `assets/models/` 目录下是否有模型文件
- 如果没有模型文件，程序会自动使用占位几何体（圆柱体）

### 按钮无响应
- 检查浏览器控制台是否有 JavaScript 错误
- 确保所有脚本文件都已正确加载

## 浏览器兼容性

- Chrome/Edge（推荐）
- Firefox
- Safari

需要支持 WebGL 和 ES6 模块的现代浏览器。
